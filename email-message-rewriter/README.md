# Email Message Rewriter

## Overview
Takes a rough draft of an email or message and rewrites it to be more professional, clear, and well-toned.

## How to use
Tell the agent:
> "Rewrite this to my boss to be more professional: 'hey I cant finish the report today I need more time sorry.'"

The agent will consult `SKILL.md` in this directory to draft the perfect communication.
