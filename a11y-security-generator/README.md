# Accessibility & Security Test Generator

## Overview
Easily upgrade your Selenium Java tests to include accessibility compliance (axe-core) and basic OWASP security validations.

## How to use
Provide your existing test or a description of your UI form/API:
> "Generate a11y and security tests for a User Profile update form containing a First Name, Last Name, and Bio text area."

The skill will construct the Selenium code required to scan for WCAG violations and test common injection vulnerabilities.
