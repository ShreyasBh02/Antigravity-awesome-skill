# Flaky Test Healer & Refactorer

## Overview
Maintain stable, reliable Selenium WebDriver (Java) UI tests by removing flakiness, hardcoded waits, and bad locators automatically.

## How to use
Just paste your failing or unstable Selenium Java test script and say:
> "This test fails randomly on the login button click. Please heal it."

The skill will analyze the waits, locators, and state management, providing a hardened version of the test.
