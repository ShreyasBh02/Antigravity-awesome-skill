# Daily Planner Prioritizer

## Overview
Organizes your daily tasks and helps smartly prioritize them based on urgency and overall goals.

## How to use
Tell the agent:
> "Help me plan my day: I need to review 3 PRs, attend a sync at 2 PM, fix the login bug, and read a new spec."

The agent will refer to `SKILL.md` in this directory to optimize your daily agenda.
