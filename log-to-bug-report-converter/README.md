# Log-to-Bug-Report Converter

## Overview
Turn an unreadable wall of red text into a pristine, actionable bug ticket instantly.

## How to use
Simply paste the massive stack trace, failing test console output, or server log snippet, and say:
> "Convert this log failure into a bug report."

The skill will filter the noise, extract the core issue, and format a beautiful Jira/GitHub/Azure DevOps-compatible markdown ticket.
