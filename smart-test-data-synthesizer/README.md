# Smart Test Data Synthesizer

## Overview
Need a user who has three cancelled orders and one active subscription but an expired credit card? This skill synthesizes complex relational test data for you.

## How to use
Describe your data state:
> "Generate test data for an e-commerce checkout. I need a User, Product, and Cart schema. Output as SQL inserts and Java DataFaker code."

The skill will intelligently map the relationships and generate the underlying code and scripts to build that exact state.
