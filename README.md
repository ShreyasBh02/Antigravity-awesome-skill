# My Skills Collection

This repository contains a set of AI skills originally packaged as `.skill` archive files. They have now been extracted into individual folders. Each folder contains the actual `SKILL.md` file along with its own `README.md` explaining how to use it.

## Available Skills

| Skill Folder | Purpose / Description |
|-----------|-------------|
| [api-test-script-builder/](api-test-script-builder/README.md) | Generates ready-to-run API test scripts from endpoint descriptions (e.g. Swagger, Postman, plain English). |
| [bug-report-writer/](bug-report-writer/README.md) | Crafts clear, professional bug reports with steps to reproduce, expected vs. actual outcomes. |
| [daily-planner-prioritizer/](daily-planner-prioritizer/README.md) | Organizes daily tasks and helps to intelligently prioritize them based on urgency and goals. |
| [email-message-rewriter/](email-message-rewriter/README.md) | Rewrites emails or messages to adjust their tone, clarity, formatting, and professionalism. |
| [recipe-grocery-list/](recipe-grocery-list/README.md) | Processes recipes to extract and structure actionable grocery shopping lists categorized by items. |
| [test-case-generator/](test-case-generator/README.md) | Automatically generates thorough, edge-case-inclusive test cases for features and functions. |

## How to Use These Skills
Navigate to any of the specific skill folders listed above. Inside, you will find a `README.md` file that explains how to activate that skill with an AI, and the `SKILL.md` file which contains the raw behavioral instructions.
