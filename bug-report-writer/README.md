# Bug Report Writer

## Overview
Crafts clear, professional bug reports with steps to reproduce, expected vs. actual outcomes from simple descriptions or logs.

## How to use
Tell the agent:
> "Write a bug report for when I click checkout and the page crashes: the console says 'ReferenceError: cart is not defined'."

The agent will use the `SKILL.md` file in this directory to format a ready-to-use bug ticket.
